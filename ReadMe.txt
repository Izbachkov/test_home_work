Мой вариант решения практического задания № 8 курса "Серверные веб-технологии и системы управления контентом" (ИТМО)
на платформе "Открытое образование" (осень 2020 года).

Задача: перейти по заданному URL, нажать кнопку с id = "bt" и считать значение, которое появится в поле с id = "inp"

Для тестирования предоставлен адрес:
https://kodaktor.ru/g/80b5cdf
Также для тестов можно использовать:
https://github.com/Izbachkov/week9

В курсе для развертывания приложений рекомендуется использовать heroku и repl.it, однако они не позволяют развернуть на
них версию Chrome.
Можно отказаться от использования указанных платформ. Развернуть приложение локально, установить Chrome, указывать его
адрес в параметрах используя puppeteer как в лекции, а затем через прокси перенаправлять запросы.
Или использовать библиотеку zombie. Я пробовал puppeteer и selenium-webdriver, не выходит - не дают платформы развернуть браузер.

При использовании zombie нет проблем запустить браузер, перейти по URL, щелкнуть кнопку.
Проблема в том, что в поле ввода не появляется значение.

got = await browser.text('#bt').value;

- работать не будет, если использовать предоставленный тестовый адрес.
Там за это отвечает строка разметки:

<button onclick="this.previousSibling.value='0.8862481722945399'" id="bt">=)</button>

Т.е. визуально все норм, но программно считать значение не получается.
Приходится жульничать:

got = await browser.html('#bt');
hahaha = got.slice(45,63); 

Считывать строку до события, а послу делать из нее срез.
